## üìä **Business Logic Review**

### **Waste Calculation Accuracy**

**Current Method:**
```python
quantity_with_waste = quantity_net * waste_factor  # e.g., 4.5m √ó 1.05 = 4.725m
vendor_quantity = ceil(4.725 √∑ 6.0) √ó 6.0 = 6.0m  # Buy 6m
effective_waste = (6.0 - 4.5) √∑ 4.5 = 33.3%       # Actual waste is higher
```

**Issue**: Waste factor applied before vendor units, resulting in **double waste consideration**.

**Improved Method:**
```python
vendor_quantity = ceil(quantity_net √∑ 6.0) √ó 6.0 = 6.0m  # Buy 6m (no waste factor yet)
usable_waste = min(6.0 - 4.5, 4.5 √ó waste_factor)       # Consider reusable waste
final_cost = calculate_with_realistic_waste(vendor_quantity, usable_waste)
```
